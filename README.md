# StreamPlus

An efficient multi-step onboarding app that makes user registration simple and dynamic. Itâ€™s built with Laravel and Livewire

## Installation

### 1. Clone the Repository
```bash
git clone https://github.com/mohamad-zatar/stream-plus.git
```
### 2. Install Dependencies
```bash
composer install
```
### 3. Configure the Environment
```bash
cp .env.example .env
```
Update the .env file with:DB_DATABASE | DB_USERNAME | DB_PASSWORD

### 4. Generate the Application Key
```bash
php artisan key:generate
```
### 5. Run Migrations
```bash
php artisan migrate
```
### 6. install and build your NPM dependencies
```bash
npm install && npm run build
```

### 7. Start the Development Server
```bash
php artisan serve
```
